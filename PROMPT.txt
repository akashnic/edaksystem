🔥 BUSINESS LOGIC
1. Create Receive Entry
Default status = NEW

2. Dispatch Entry Creation
Must link using L_ID
Prevent duplicate dispatch
Update Receive.current_status → DISPATCHED

3. Mark as Received
Update dispatch.status → RECEIVED
Set received_at timestamp
Update Receive.current_status → RECEIVED
